Установка с использованием .deb

ШАГ 1. Скачивание и установка .deb

Сначала скачайте пакет .deb с последней версией Flowblade отсюда.

Для установки дважды щёлкните по пакету .deb.

В некоторых системах двойной щелчок может не сработать (на Ubuntu иногда ставится старая версия из репозитория), в этом случае вам придётся установить пакет .deb с помощью терминала:

  • Откройте терминал в папке, в которой был сохранён загруженный пакет .deb. Задайте команду:

    sudo dpkg -i ./flowblade-1.16.0-1_all.deb

ШАГ 2 (ДОПОЛНИТЕЛЬНЫЙ). Дополнительные команды в терминале

Возможно, вам понадобится задать дополнительные команды в терминале:

  • Для установки всех зависимостей:
    • sudo apt-get install -f

Релиз был протестирован на: Ubuntu 16.04, Ubuntu 18.04, Linux Mint 18, Linux Mint 19 и Debian 8. Также должен работать на всех последующих дистрибутивах Debian.

Установка из репозитория вашего дистрибутива

Самый простой способ установки Flowblade — это использовать версию из репозитория вашего дистрибутива. Недостатком является то, что доступная версия может быть устаревшей. Если в репозитории вашего дистрибутива используется старая версия Flowblade или отсутствует вовсе, сообщите об этом его разработчиком.

Установка актуальной версии из репозитория ROSA Desktop Fresh R10

  • Откройте терминал и задайте команду для обновления списков программ:

    sudo urpmi.update -a

  • Затем ещё одну команду для установки Flowblade:

    sudo urpmi flowblade

Установка из исходного кода

Flowblade в настоящее время на 100% является скриптовым приложением, и все зависимости должны быть доступны в популярных дистрибутивах, поэтому в большинстве случаев Flowblade можно запустить без компиляции.

Получить дополнительную информацию можно здесь.

Установка разрабатываемой версии

Flowblade в настоящее время на 100% является скриптовым приложением, и все зависимости должны быть доступны в популярных дистрибутивах, поэтому в большинстве случаев Flowblade можно запустить без компиляции.

Для дистрибутива ROSA Desktop Fresh R10 создан репозиторий с разрабатываемой версией. Для его добавления откройте терминал и задайте одну из команд:

  • для 64 битных систем:

    • sudo urpmi.addmedia smolnp_personal http://abf-downloads.rosalinux.ru/smolnp_personal/repository/rosa2016.1/x86_64/main/release

  • для 32 битных систем:
    • urpmi.addmedia smolnp_personal http://abf-downloads.rosalinux.ru/smolnp_personal/repository/rosa2016.1/i586/main/release
