Установка

Установка с использованием .deb

Сначала скачайте пакет .deb с последней версией Flowblade отсюда.

Для установки дважды щёлкните по пакету .deb.

В некоторых системах двойной щелчок может не сработать (на Ubuntu иногда ставится старая версия из репозитория), в этом случае вам придётся установить пакет .deb с помощью терминала:

  • Откройте терминал в папке, в которой был сохранён загруженный пакет .deb и задайте команду:

    sudo apt install ./flowblade-2.2.0-1_all.deb

Релиз был протестирован на: Ubuntu 16.04, Ubuntu 18.04, Linux Mint 18, Linux Mint 19 и Debian 8. Также должен работать на всех последующих дистрибутивах Debian.

Установка из репозитория вашего дистрибутива

Самый простой способ установки Flowblade — это использовать версию из репозитория вашего дистрибутива. Недостатком является то, что доступная версия может быть устаревшей. Если в репозитории вашего дистрибутива используется старая версия Flowblade или отсутствует вовсе, сообщите об этом его разработчиком.

Ubuntu, Debian и Linux Mint

  • Откройте терминал и задайте команду:

    sudo apt-get install flowblade

ROSA Desktop Fresh, OpenMandriva

  • Откройте терминал и задайте команду:

    sudo urpmi flowblade

Archlinux

  • Перейдите на страницу AUR или задайте команду в терминале:

    yaourt -S flowblade

Установка разрабатываемой версии

Разрабатываемая версия может быть нестабильной и/или иметь новые зависимости.

Archlinux

  • Перейдите на страницу AUR или задайте команду в терминале:

    yaourt -S flowblade-git

Запуск без установки (из исходного кода)

Flowblade в настоящее время на 100% является скриптовым приложением, а все зависимости должны быть доступны в популярных дистрибутивах. Поэтому, в большинстве случаев, как стабильную так и разрабатываемую версии Flowblade можно запустить без компиляции.

  1. В начале скачайте архив с исходным кодом с нужной стабильной версией отсюда и распакуйте архив в любую папку. Если вы хотите попробовать разрабатываемую версию выполните следующие действия с помощью терминала:

    • Установите Git в вашей системе (к примеру в Ubuntu)
      • sudo apt-get install git
    • Скачайте Flowblade c Git репозитория
      • git clone https://github.com/jliljebl/flowblade.git
  2. Установите зависимости. Информацию о необходимых зависимостях можно посмотреть в разделе Зависимости.
  3. Если в вашей системе Flowblade был установлен ранее, все необходимые зависимости также должны были установиться. Но помним, что могли быть добавлены новые.
  4. Запустите файл flowblade, который должен находится в распакованной или скачанной папке по адресу .../flowblade-trunk/flowblade
    • Обратите внимание: если в вашей системе уже имеется установленная версия Flowblade, для запуска сценария вам потребуется указать полный путь к запускаемой версии или открыть терминал в папке, содержащей сценарий запуска и задать команду ./flowblade, во избежании запуска установленной версии.

    Windows и OS X не поддерживаются. Разработчики Flowblade сосредоточены на предоставлении наилучшей поддержки в Linux и в других свободных платформах.
