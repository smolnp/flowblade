Установка

ПРИМЕЧАНИЕ: Новые версии Flowblade требуют MLT 6.18, который недоступен, например, в Ubuntu 18.04.

Установка с использованием .deb

Сначала скачайте пакет .deb с последней версией Flowblade отсюда.

Для установки дважды щёлкните по пакету .deb.

В некоторых системах двойной щелчок может не сработать (на Ubuntu иногда ставится старая версия из репозитория), в этом случае вам придётся установить пакет .deb с помощью терминала:

  • Откройте терминал в папке, в которой был сохранён загруженный пакет .deb и задайте команду:

    sudo apt install ./flowblade-2.6.3-1_all.deb

Установка из репозитория вашего дистрибутива

Самый простой способ установки Flowblade — это установка версии из репозитория вашего дистрибутива. Основным недостатком такой установки может стать устаревшая версия видеоредактора. Если в репозитории вашего дистрибутива находится старая версия Flowblade или отсутствует вовсе, сообщите об этом его разработчикам.

Arch Linux

  • Включите репозиторий сообщества в /etc/pacman.conf:

    [community]
    Include = /etc/pacman.d/mirrorlist

  • Задайте команду установки:

    pacman -Syu flowblade

Debian, Ubuntu и Linux Mint

  • Обновите список репозиториев:

    sudo apt update

  • Задайте команду установки:

    sudo apt install flowblade

Fedora

  • Загрузите последнюю версию rpmfusion-free-release-stable с:

    http://download1.rpmfusion.org/free/fedora/

  • Установите rpmfusion-free-release-stable rpm:

    sudo rpm -Uvh rpmfusion-free-release-stable*rpm

  • Задайте команду установки:

    sudo dnf install flowblade

OpenMandriva

  • Включите репозиторий в приложении выбора репозитория (om-repo-picker)

  • Задайте команду установки:

    dnf install flowblade

ROSA Desktop Fresh

  • Откройте терминал и задайте команду:

    sudo urpmi flowblade

Установка через Flatpak

Flowblade также можно установить через Flatpak.

Если в вашем дистрибутиве присутствует приложение Gnome SOFTWARE, вы можете установить Flowblade из него.

Можно через терминал:

  • Команда установки

    flatpak install flathub io.github.jliljebl.Flowblade

  • Команда запуска

    flatpak run io.github.jliljebl.Flowblade
ПРИМЕЧАНИЕ: версия в Flatpak может отставать от версии из основного репозитория.

Запуск без установки (из исходного кода)

Flowblade в настоящее время на 100% является скриптовым приложением, а все зависимости должны быть доступны в популярных дистрибутивах. Поэтому, в большинстве случаев, как стабильную так и разрабатываемую версии Flowblade можно запустить без компиляции.

  • В начале скачайте архив с исходным кодом с нужной стабильной версией отсюда и распакуйте архив в любую папку. Если вы хотите попробовать разрабатываемую версию выполните следующие действия с помощью терминала:

  • Установите Git в вашей системе (к примеру в Ubuntu)

    sudo apt-get install git

  • Скачайте Flowblade c Git репозитория

    git clone https://github.com/jliljebl/flowblade.git
  • Установите зависимости. Информацию о необходимых зависимостях можно посмотреть в разделе Зависимости.

Если в вашей системе Flowblade был установлен ранее, все необходимые зависимости также должны были установиться. Но помним, что могли быть добавлены новые.

  • Запустите файл flowblade, который должен находится в распакованной или скачанной папке по адресу .../flowblade-trunk/flowblade.

ОБРАТИТЕ ВНИМАНИЕ: если в вашей системе Flowblade уже установлен, для запуска сценария вам потребуется указать полный путь к запускаемой версии или открыть терминал в папке, содержащей сценарий запуска и задать команду ./flowblade, во избежании запуска установленной версии.

Windows и OS X не поддерживаются. Разработчики Flowblade сосредоточены на предоставлении наилучшей поддержки в Linux и в других свободных платформах.
